# **NSW-Traffic-Analysis**

**# NSW Road Traffic Analysis and Prediction 🚦**



**## Overview**

**This project analyzes road traffic volume data from New South Wales (NSW), Australia.**  

**The dataset is sourced from the Transport for NSW Open Data Portal.**  



**The objective of this project is to:**

**- Clean and preprocess traffic data**

**- Perform exploratory data analysis (EDA)**

**- Visualize traffic patterns**

**- Build machine learning models to predict traffic volumes**



**## Dataset**

**- Source: Transport for NSW Open Data Portal**  

**- Dataset: NSW Road Traffic Volume Counts**  



**## Tools \& Technologies**

**- Python**

**- Pandas**

**- Matplotlib**

**- Seaborn**

**- Scikit-learn**

**- Jupyter Notebook**



**## Project Workflow**

**1. Data Collection**  

**2. Data Cleaning**  

**3. Exploratory Data Analysis (EDA)**  

**4. Data Visualization**  

**5. Machine Learning Modeling**  

**6. Model Evaluation**  



**## Exploratory Data Analysis**

**- Analyzed yearly traffic trends**  

**- Identified busiest roads**  

**- Visualized traffic distribution**  



**## Machine Learning Models**

**- Decision Tree Regressor**  

**- Random Forest Regressor**  



**### Model Evaluation**

**- Used Mean Squared Error (MSE)**  

**- Random Forest performed better due to ensemble learning**  



**## Key Insights**

**- Traffic volume shows variation across years**  

**- Certain roads consistently have higher traffic**  

**- Machine learning models can reasonably predict traffic trends**
